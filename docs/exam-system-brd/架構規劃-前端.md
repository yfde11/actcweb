## 1. 技術堆疊

### 1.1 核心技術

| 層級 | 技術 |
|------|------|
| 框架 | Alpine.js (CDN) |
| 樣式 | Tailwind CSS (CDN) |
| 運行環境 | 瀏覽器（原生 JS） |
| 後端通訊 | Fetch API |
| 檔案上傳 | FormData + multer |

### 1.2 技術決策

| 決策 | 選擇 | 原因 | 取捨 |
|------|------|------|------|
| Alpine.js 而非 React/Vue | 輕量、無建置、CDN 引入 | 不適合大型 SPA |
| Tailwind CSS | 快速開發、一致性、CDN | 客製化受限 |
| Fetch API | 原生支援、無依賴 | 無攔截器（需手動處理） |

## 3. Alpine.js 組件設計

### 3.1 主組件：questionBankTab()

```javascript
function questionBankTab() {
    return {
        // === 狀態 ===
        questions: [],
        domains: [
            { id: 1, name: 'Security and Risk Management' },
            { id: 2, name: 'Asset Security' },
            { id: 3, name: 'Security Architecture and Engineering' },
            { id: 4, name: 'Communication and Network Security' },
            { id: 5, name: 'Identity and Access Management' },
            { id: 6, name: 'Security Assessment and Testing' },
            { id: 7, name: 'Security Operations' },
            { id: 8, name: 'Software Development Security' }
        ],
        selectedDomain: null,
        searchQuery: '',
        page: 1,
        limit: 20,
        total: 0,
        totalPages: 1,

        // === Modal 狀態 ===
        showCreateModal: false,
        showEditModal: false,
        showImportModal: false,
        showGenerateExamModal: false,
        showStatisticsModal: false,
        showUsageModal: false,

        editingQuestion: null,
        selectedQuestions: [],  // 複選框
        
        // === 表單 ===
        formData: {
            type: 'multiple_choice',
            domain: 1,
            content: '',
            options: [
                { text: '', label: 'A' },
                { text: '', label: 'B' },
                { text: '', label: 'C' },
                { text: '', label: 'D' }
            ],
            correctOptionIndex: 0,
            correctBoolean: true,
            correctAnswers: [''],
            acceptableAnswers: [''],
            points: 1,
            difficulty: 'easy',
            explanation: ''
        },

        // === 初始化 ===
        async init() {
            await this.loadQuestions();
        },

        // === 載入題目 ===
        async loadQuestions(domain = null, search = '') {
            this.loading = true;
            try {
                const query = new URLSearchParams();
                if (domain) query.append('domain', domain);
                if (search) query.append('search', search);
                query.append('page', this.page);
                query.append('limit', this.limit);

                const res = await fetch(`/api/question-bank?${query}`, {
                    headers: { 'Authorization': `Bearer ${localStorage.getItem('adminToken')}` }
                });
                if (res.ok) {
                    const data = await res.json();
                    this.questions = data.data || [];
                    this.total = data.pagination.total;
                    this.totalPages = data.pagination.totalPages;
                }
            } catch (e) {
                console.error('Load questions error:', e);
            } finally {
                this.loading = false;
            }
        },

        // === CRUD ===
        async createQuestion() {
            try {
                const res = await fetch('/api/question-bank', {
                    method: 'POST',
                    headers: {
                        'Authorization': `Bearer ${localStorage.getItem('adminToken')}`,
                        'Content-Type': 'application/json'
                    },
                    body: JSON.stringify(this.formData)
                });
                if (res.ok) {
                    this.showCreateModal = false;
                    await this.loadQuestions(this.selectedDomain);
                } else {
                    alert('建立失敗');
                }
            } catch (e) {
                console.error('Create question error:', e);
            }
        },

        async updateQuestion(id) {
            try {
                const res = await fetch(`/api/question-bank/${id}`, {
                    method: 'PUT',
                    headers: {
                        'Authorization': `Bearer ${localStorage.getItem('adminToken')}`,
                        'Content-Type': 'application/json'
                    },
                    body: JSON.stringify(this.formData)
                });
                if (res.ok) {
                    this.showEditModal = false;
                    await this.loadQuestions(this.selectedDomain);
                }
            } catch (e) {
                console.error('Update question error:', e);
            }
        },

        async deleteQuestion(id) {
            if (!confirm('確定要刪除這個題目嗎？')) return;
            try {
                const res = await fetch(`/api/question-bank/${id}`, {
                    method: 'DELETE',
                    headers: { 'Authorization': `Bearer ${localStorage.getItem('adminToken')}` }
                });
                if (res.ok) {
                    await this.loadQuestions(this.selectedDomain);
                }
            } catch (e) {
                console.error('Delete question error:', e);
            }
        },

        // === 批量匯入 ===
        async importCSV(file) {
            const formData = new FormData();
            formData.append('file', file);

            try {
                const res = await fetch('/api/question-bank/import', {
                    method: 'POST',
                    headers: { 'Authorization': `Bearer ${localStorage.getItem('adminToken')}` },
                    body: formData
                });
                const data = await res.json();
                alert(`匯入成功 ${data.data.imported} 題，失敗 ${data.data.errors.length} 題`);
                this.showImportModal = false;
                await this.loadQuestions();
            } catch (e) {
                console.error('Import error:', e);
            }
        },

        // === 生成考試 ===
        async generateExam(mode, params) {
            try {
                const res = await fetch('/api/exams/from-bank', {
                    method: 'POST',
                    headers: {
                        'Authorization': `Bearer ${localStorage.getItem('adminToken')}`,
                        'Content-Type': 'application/json'
                    },
                    body: JSON.stringify(params)
                });
                if (res.ok) {
                    alert('考試生成成功');
                    this.showGenerateExamModal = false;
                }
            } catch (e) {
                console.error('Generate exam error:', e);
            }
        },

        // === 統計 ===
        async loadStatistics() {
            try {
                const res = await fetch('/api/question-bank/statistics', {
                    headers: { 'Authorization': `Bearer ${localStorage.getItem('adminToken')}` }
                });
                if (res.ok) {
                    this.statistics = await res.json();
                    this.showStatisticsModal = true;
                }
            } catch (e) {
                console.error('Load statistics error:', e);
            }
        },

        // === 篩選 ===
        filterByDomain(domain) {
            this.selectedDomain = domain;
            this.page = 1;
            this.loadQuestions(domain);
        },

        search() {
            this.page = 1;
            this.loadQuestions(this.selectedDomain, this.searchQuery);
        }
    }
}
```

## 6. 效能優化

### 6.1 分頁加載

```javascript
// 使用 Alpine.js 的 $watch 或 @change 事件
page: 1,

watchPage() {
    this.page = 1;  // 篩選時重設頁碼
    this.loadQuestions();
}
```

### 6.2 搜尋防抖

```javascript
searchQuery: '',
searchTimer: null,

search() {
    clearTimeout(this.searchTimer);
    this.searchTimer = setTimeout(() => {
        this.page = 1;
        this.loadQuestions(this.selectedDomain, this.searchQuery);
    }, 300);  // 300ms 防抖
}
```

### 6.3 複選框批次操作

```javascript
selectedQuestions: [],

toggleAll() {
    if (this.selectedQuestions.length === this.questions.length) {
        this.selectedQuestions = [];
    } else {
        this.selectedQuestions = this.questions.map(q => q._id);
    }
}
```
